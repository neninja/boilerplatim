# boilerplatim

Aka boilerplatinho.

Leia `:help boilerplatim`.

## Installation

**Using Packer**

```lua
use {
    'nenitf/boilerplatim',
    requires = {
      'nvim-lua/plenary.nvim',
    },
    config = function()
      require('boilerplatim').setup()
    end,
}
```
